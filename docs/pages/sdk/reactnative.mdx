# React Naive SDK for Embedded Wallet

The Sequence React Native Embedded Wallet SDK provides complete Sequence [Embedded Wallet](/solutions/wallets/embedded-wallet/overview) and [Indexer](/api/indexer/overview) integration for React Native applications.

You can see a full example of how to use the SDK in the [Embedded Wallet React Native Demo](https://github.com/0xsequence/demo-waas-react-native) repository.
