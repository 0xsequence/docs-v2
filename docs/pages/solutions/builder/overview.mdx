# Welcome 

import { Card, CardGrid } from "../../../components/CardGrid";

[Sequence Builder](https://sequence.build/) is your gateway to the Sequence platform. An all-inclusive, web3 game development dashboard for managing your projects, API keys, wallet instantiation, contract and marketplace deployments and more. It also provides user access management so multiple people in your team can easily work on the same project. In addition to these powerful features, Builder also provides easy to use interfaces for managing sponsored wallets with your gas tank, and integration tools for Sequence Kit, Node Gateway and Indexer.

To learn more about how to leverage the powerful features that Sequence has to offer, follow the walkthroughs below:
